# Practica 1 Parcial 2

## Objetivo
El objetivo de esta practica es desarrollar las competencias para el manejo de PDO en php y su correcta utilizacion.

## Tecnologias utilizadas
- PHP 8+
- MySQL
- PDO

## Instrucciones de Ejecucion
Para ejecutarlo localmente abrimos el panel de administracion de XAMPP y los archivos que hayamos desarrollado los metemos en la carpeta htdocs de la ruta raiz de XAMPP.

## Funcionamiento
En este caso crea una pequeña interfaz en la pagina web donde podemos hacer los procesos de CRUD que es Crear, Leer, Actualizar y Eliminar, simulando un inventario.
