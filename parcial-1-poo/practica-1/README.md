## PRÁCTICA DE LABORATORIO 1: CREACIÓN DE CLASES Y ENCAPSULAMIENTO EN PHP

### Objetivo:
El objetivo de la practica es poner en practica los conocimientos de PHP que hemos estado viendo en la primera unidad para su utilizacion en Programacion Orientada a Objetos (POO).

### Clase Usuario:
En esta clase se registran los datos de correo y nombre del usuario para su utilizacion en otras funcionalidades.

### Instrucciones de ejecucion:
Para ejecutar el codigo primeramente se requiere de PHP Intelephense y PHP Server, una vez descargado esos pluggins la ejecucion es la siguiente das click derecho al archivo de index.php y como opcion le damos en PHP Server: Serve Project y listo se desplegara la informacion que tengas en el index llamando a los metodos de la clase usuario en el archivo de usuarios.php
